# Profession Pages

## One-liner

Profession-specific landing pages (19) with research-backed NYSED content, SEO, sitemap, and nav.

## Status

**Live.** Includes CPA, Veterinarian, Optometrist, Pharmacist, Podiatrist (Mar 2026 wave) plus earlier set.

## Scope boundary

We handle **PLLC formation only** (NYSED pre-approval package, naming, Articles, publication, bank-ready docs). We do **not** coordinate with state boards, firm registration (e.g. CPA Form 6R), or dual-registration — describe those as customer-owned requirements.

## Key paths

- App routes under `web/src/app/` for each profession
- Shared SEO / breadcrumb / nav patterns

## Gotchas

- Content must stay within formation scope (see boundary above)
- New pages need metadata, structured data, sitemap, nav
